# Media11Hub – Website Podcast

Acest proiect conține toate resursele necesare pentru rularea și dezvoltarea website-ului **Media11Hub**, un proiect de podcast dedicat **Școlii Gimnaziale Nr. 11 Buzău**.

Website-ul este conceput pentru prezentarea episoadelor de podcast într-o interfață modernă și profesională, cu redare video direct pe site, fără utilizarea platformelor externe, oferind o experiență clară și accesibilă pentru elevi, profesori și vizitatori.

## Mențiune
Acest website **nu este unul oficial al școlii**.  
Proiect creat în scop educațional și demonstrativ.
